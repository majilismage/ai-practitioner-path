Course Structure

This programme is structured as 5 progressive missions, each building on the previous:

  • Mission 1: Foundation - Master the fundamentals of effective AI collaboration
  • Mission 2: Mental Models - Understand what AI can and can't do strategically
  • Mission 3: Professional Application - Deploy AI tools in real work contexts
  • Mission 4: Agent Systems - Build AI agents that work independently
  • Mission 5: Production Systems - Scale and maintain AI solutions safely

Each mission contains focused, practical modules designed around real-world application rather than theoretical knowledge. You'll build actual systems, not just learn about them.

Prerequisites

What You'll Need
  • Professional experience in any domain (technical or non-technical)
  • Access to Claude Pro or API credits ($20-40/month recommended)
  • Willingness to experiment with real work projects
  • Basic comfort with web browsers and software installation
